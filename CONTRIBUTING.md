# 🤝 Guía de Contribución

¡Gracias por interesarte en contribuir a Repuestos Motos Store!

## Cómo Contribuir

1. **Fork** el repositorio
2. **Clona** tu fork: `git clone https://github.com/tu-usuario/repuestos-motos-store.git`
3. **Crea una rama**: `git checkout -b feature/tu-feature`
4. **Haz cambios** y **commits**: `git commit -m "Agrega: descripción"`
5. **Push** a tu rama: `git push origin feature/tu-feature`
6. **Abre un Pull Request**

## Convenciones de Código

### Commits
```
feat: Agrega nueva funcionalidad
fix: Corrige un bug
docs: Actualiza documentación
style: Cambios de formato
refactor: Refactoriza código
test: Agrega tests
```

### Nombres de Ramas
```
feature/nombre-feature
bugfix/descripcion-bug
docs/actualizar-readme
```

## Estándares

- Usa **camelCase** para variables y funciones
- Usa **PascalCase** para componentes React
- Comenta código complejo
- Escribe commits descriptivos
- Prueba antes de hacer PR

## Reportar Bugs

1. Ve a [Issues](https://github.com/pampalogistica3-beep/repuestos-motos-store/issues)
2. Click en "New Issue"
3. Describe el problema con detalles
4. Incluye pasos para reproducir

## Solicitar Features

1. Abre un [Issue](https://github.com/pampalogistica3-beep/repuestos-motos-store/issues)
2. Etiqueta como "enhancement"
3. Describe la feature en detalle
